# Single user

To get started using Apollo as a single user, you'll need access to a JBrowse 2
instance with your data.

## JBrowse 2 on the Web

If the genome you're working on already has a JBrowse 2 version hosted
somewhere, such as on a model organism database website, you can use Apollo on
that site. See this guide for an example of how to do this.

<a href="single-user/jbrowse-web" className="button button--primary button--lg">Using
Apollo on JBrowse 2 on the Web</a>

## JBrowse 2 Desktop

If you want to use Apollo with data you have on your own computer, we suggest
using JBrowse 2 Desktop. See the following for guide for step-by-step
instructions on how to set this up.

<a href="single-user/jbrowse-desktop" className="button button--primary button--lg">Using
Apollo on a JBrowse Desktop</a>
