# `apollo help`

- [`apollo help [COMMAND]`](#apollo-help-command)

## `apollo help [COMMAND]`

Display help for apollo.

```
USAGE
  $ apollo help [COMMAND...] [-n]

ARGUMENTS
  [COMMAND...]  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for apollo.
```
