# Multiple users

In a multi-user Apollo installation, users connect to an **Apollo Collaboration
Server** and work on a shared copy of the data. This enables **Real-time
collaboration** between users. Multiple annotators can work on the same assembly
at the same time. Changes made by one user are immediately visible to all other
connected users, so teams can divide annotation work across chromosomes or gene
families without worrying about merging divergent copies later.

## Trying out multi-user Apollo

If you want to set up Apollo for multiple users, or if you just want to try it
out, we suggest first going through our guide to set up a local demo of a full
multi-user Apollo setup. This will allow you to get familiar with the various
pieces of an Apollo installation and try things out like adding assemblies and
bulk importing annotations.
